    Description

    LibPKMN is the most universal library available for Pokémon game save editing and strategy analysis. It is capable of manipulating save files from all Generation 1-5 games (22 games released 1996-2013). Users can generate Pokémon on their PC and export them into their teams. Even without a game, the library can be used for detailed team and strategy analysis.

    This library is designed to be as portable as possible. Using the CMake build system for platform-independence, LibPKMN can be compiled with any modern version of GCC, Clang, or Microsoft Visual Studio C++. Portability is further extended through the use of SWIG, which generates Python, C#, and Java wrappers to allow further development.
